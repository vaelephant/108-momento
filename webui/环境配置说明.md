# 环境配置说明

## 🔧 必需的环境变量

在 `webui` 目录下创建 `.env.local` 文件：

```bash
# API配置
NEXT_PUBLIC_API_URL=http://localhost:8003

# 应用配置
NEXT_PUBLIC_APP_NAME=Momento AI Photo Management
```

## 🚀 启动步骤

### 1. 启动后端服务
```bash
cd /Users/yzm/code/108-momento/server
python start.py
```

### 2. 启动前端服务
```bash
cd /Users/yzm/code/108-momento/webui
npm run dev
```
